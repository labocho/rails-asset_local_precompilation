# AssetTasks
Short description and motivation.

## Usage

```ruby
rails g asset_tasks
```

Edit `Capfile`

```ruby
require "capistrano/local_compile"
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'asset_tasks'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install asset_tasks
```
